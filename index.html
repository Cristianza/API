API Documentation (Examples):

Search by several criteria:

Using the following structure: 

/consultas/[Criteria]/[Name, state, hotel type or size]

By hotel name:

Sample URL:

http://localhost/API/consultas/hotel_name/ITC%20HOTEL%20KAKATIYA

By state:

Sample URL:

http://localhost/API/consultas/state/KERALA

By hotel type:

Sample URL:

http://localhost/API/consultas/type/5%20star%20deluxe

By hotel size (# of rooms):

Sample URL:

http://localhost/API/consultas/size/large

Check for hotel availability:

Using the following structure: 

/disponibilidad/[Start Date (YYYY/MM/DD)]/[End Date (YYYY/MM/DD)]/[State]

Sample URL:

http://localhost/API/disponibilidad/2018-11-24_2018-11-27_KERALA

Make a reservation for an hotel (could be more than one):

Using the following structure: 

/reservacion/[Hotel ID]/[User ID]/[Start Date (YYYY/MM/DD)]/[End Date (YYYY/MM/DD)]/[# of reservations]

Sample URL:

http://localhost/API/reservacion/1_2_2018-11-24_2018-11-27_1

Using an API testing tool, for example Postman:

Create a new user:

Select 

POST
in Postman and then write the URL following this structure: 

/usuario/[User ID]

Sample URL:

http://localhost/API/usuario/add

And then add the different variables to be used:

  • name:

    User's first name

  • last_name:

    User's last name

  • address:

    User's address

  • email:

    User's email

  • password.

    User's account password

Update user's data:

Select 

PUT
in Postman and then write the URL following this structure: 

/usuario/[User ID]

Sample URL:

http://localhost/API/usuario/update/1

And then add the different variables to be used:

  • id:

    User ID

  • name:

    User's first name

  • last_name:

    User's last name

  • address:

    User's address

  • email:

    User's email

  • password.

    User's account password

Create a new API key:

Select 

PUT
in Postman and then write the URL following this structure: 

/createkey

Sample URL:

http://localhost/API/createkey

And then add the different variables to be used:

  • contact_name:

    Company's contact name

  • company:

    Company's name

  • email:

    Company's email

Post a new hotel:

Select 

POST
in Postman and then write the URL following this structure: 

/hotel/add

Sample URL:

http://localhost/API/hotel/add

And then add the different variables to be used:

  • name:

    Hotel's name

  • address:

    Hotel's address

  • state:

    State where the hotel is placed

  • phone:

    Hotel's phone number

  • fax:

    Hotel's fax number

  • email:

    Hotel's email

  • website:

    Hotel's website link

  • type:

    Hotel type

  • rooms:

    Number of rooms

Update hotel's data:

Select 

PUT
in Postman and then write the URL following this structure: 

//hotel/update/[id del hotel]

http://localhost/API/15

And then add the different variables to be used:

  • type:

    Hotel type

  • phone:

    Hotel's phone number

  • website:

    Hotel's website link

  • email:

    Hotel's email
